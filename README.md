# CrudApi
A beginner-friendly Spring Boot RESTful API project that performs Create, Read, Update, and Delete (CRUD) operations on student data using Java.
# CRUD API using Spring Boot

## Technologies Used
- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- Postman

## Features
- Create Student
- Read Student
- Update Student
- Delete Student

## API Endpoints

POST /students
GET /students
GET /students/{id}
PUT /students/{id}
DELETE /students/{id}
